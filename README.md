# facebook-navbar-clone

A clone of Facebook's dark theme nav bar

## Prerequisites

- [Node](https://nodejs.org/en/) installed

## Installation

```javascript
// Install node modules for first time
$ npm install

// start server on localhost
$ npm start
```
